# Real-Time Banking Transaction Processing System

This project is a **Real-Time Banking Transaction Processing System** built around **Kafka**, **idempotent processing**, and an **event-driven architecture**.

## Development

To run the project in development mode:

```bash
git clone https://github.com/SauravKumar-max/realtime-banking-transaction-engine.git
cd realtime-banking-transaction-engine
yarn install
yarn dev
```
